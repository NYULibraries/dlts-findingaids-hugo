Processing Information Note

Because the collection was microfilmed, no additional processing was possible. In order to create this Container List, however, all 57 reels were surveyed to identify the frame location of every folder and to fully cross-reference each reel and frame location with its original box and folder. The review also made it possible to provide dates and date ranges for each folder and to expand a number of the folder titles, highlighting, in some cases, content that might be of particular interest to Tamiment researchers.
